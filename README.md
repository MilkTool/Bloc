# Bloc 

Bloc is a low-level UI infrastructure & framework for [Pharo](http://pharo.org/).

## Installation

The following script installs Bloc intended for [Pharo 8](https://pharo.org/download):<br>

```smalltalk
Metacello new
        baseline: 'Bloc';
        repository: 'github://feenkcom/Bloc/src';
        load
```
