## Installation

The following script installs Bloc intended for [Pharo 7](https://pharo.org/download):<br>

```smalltalk
Metacello new
   baseline: 'Bloc';
   repository: 'github://feenkcom/Bloc/src';
   load
```

**More detailed installation guide is described [here](INSTALL.md)**
